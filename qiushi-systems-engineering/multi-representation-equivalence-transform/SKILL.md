# 多表述等价变换

> "拉氏变换本身只是一个'翻译'的手续，它把一个用时间变数t所描述的物理过程翻译成用变数s所描述的过程……在t的语言里用分析运算所描述的过程，用s的语言来叙述就只要用简单的代数运算。"
> —— 钱学森、宋健《工程控制论》第2章

## 核心原则

**同一系统可以用多种数学语言等价描述。当在当前表述下问题难以处理时，通过等价变换切换到另一种表述，在目标表述中求解，再变换回来验证。**

这不是"换一个工具"——是"换一个看待同一事物的视角"。等价变换保证变换前后的信息不增不减。

> 详细原著依据见《工程控制论》第2章（四种数学方法体系）、第3章（传递函数与频率特性等价）、第12章（偏微分方程与泛函分析等价）。

## 不适用场景

以下情况**不需要**调用本 skill：
- 当前工具已经高效解决——不要为了"多角度"而多角度
- 只有一种已知方法且没有理由怀疑它——别绕路
- 问题本质上是表述中立的（如逻辑推理、概念澄清）
- 已知目标表述的计算复杂度不比当前表述更低

## 何时使用

- 在某种数学框架下反复碰壁
- 微分方程难→试试代数方程；时域难→试试频域
- 随机分析复杂→试试希尔伯特空间几何化
- 需要可视化但当前是代数表述→试试相平面/频率图
- 数值计算不可行→试试解析变换

## 方法流程

### 第一步：明确当前表述

写下你现在正在用什么数学语言描述问题：
- □ 时域微分/差分方程
- □ s域/z域传递函数（代数方程）
- □ 频域（频率特性/谱密度）
- □ 状态空间（向量微分方程）
- □ 泛函空间（算子/投影/范数）
- □ 几何（相平面/参数空间）
- □ 数值（离散网格/矩阵方程）

### 第二步：列出候选等价表述与变换路径

| 源表述 → 目标表述 | 变换 | 使什么变简单 |
|-------------------|------|-------------|
| 时域微分方程 → s域 | 拉普拉斯变换 | 微分→乘法，卷积→乘积 |
| 时域差分方程 → z域 | Z变换 | 差分→乘法 |
| s域 → 频域 | 代入 s=iω | 稳定性→频率轨迹 |
| 高阶方程 → 一阶方程组 | 状态变量引入 | 数值积分，矩阵方法 |
| 状态空间 → 传递函数 | 取 F(s)=H(sI-A)^{-1}B | 消除内部状态 |
| 时域随机 → 频域功率谱 | 傅里叶变换+维纳-辛钦 | 卷积→乘积，相关→谱 |
| 向量空间 → 正交基 | 谱分解/特征展开 | 耦合→解耦 |
| 偏微分方程 → 泛函空间 | 算子半群理论 | 集中参数系统类比 |
| 代数方程 → 几何 | 参数空间映射 | 稳定性区域可视化 |
| 连续系统 → 离散系统 | 采样 + 离散化 | 数字计算 |

### 第三步：评估目标表述中"核心操作"的复杂度

不是评估整个问题的复杂度——只评估**卡住你的那个步骤**在新表述下是否变简单。

| 卡住的步骤 | 推荐切换到 |
|-----------|----------|
| 解微分方程 | s域（变成代数方程） |
| 判断稳定性 | 频域（乃奎斯特/伯德图） |
| 分析随机响应 | 频域（功率谱密度） |
| 最优滤波 | 希尔伯特空间（正交投影） |
| 参数设计 | 参数空间几何（D-划分法） |
| 多变量耦合 | 特征空间（对角化） |
| 非线性周期振荡 | 描述函数法（谐波平衡） |
| 数值迭代收敛慢 | 变分表述（梯度信息） |
| 偏微分方程控制 | 泛函分析（算子半群） |

### 第四步：在新表述中求解

直接在新表述中完成分析/设计。不需要"保持与原表述的并行思考"。

### 第五步：变换回原表述并验证

将结果翻译回原始表述（通常通过反变换公式或对照表），在原始表述中验证解的正确性。

> **变换不可逆 = 表述不等价**。如果"翻译不回去"，说明变换有信息损失，需要换另一条路径。

## 变换快捷对照

| 原表述: 操作 | 目标表述 | 变换公式 |
|-------------|---------|---------|
| 时域: 解微分方程 | s域 | L{d/dt} → s |
| 时域: 分析衰减 | s域 | 极点实部 < 0 |
| s域: 画系统行为 | 频域 | F(iω) = |F|e^{iθ} |
| 高阶标量方程 | 状态空间 | y→x, ...→x_n |
| 随机: 输出方差 | 频域 | σ² = (1/2π)∫|F(iω)|²Φ(ω)dω |
| 最优滤波解答 | 时域 | 卡尔曼递推 |

## 与其他skill的关系

**认知位置**：求解环。当在当前数学工具下卡住时启用。

**同环工具**：
- 变换得到的新表述可能暴露对偶结构 → `duality-complementarity-analysis`
- 在新表述下可用摄动法渐进求解 → `perturbation-progressive-method`
- 新表述下可能更容易形式化为优化问题 → `constrained-optimization`

**回连上游**：
- 时域↔频域切换是维纳滤波→卡尔曼滤波的理论线索 → `state-estimation`
- 表述变换的前提是理解问题的本质结构 → `first-principles-analysis`
- 变换后的结果需要回到实践检验 → `practice-cognition`

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 在新表述中从头重建整个问题 | 只变换"卡住"的那一步，其他保持不变 |
| 变换后忘记验证等价性 | 用反变换公式验证信息无损失 |
| 在多个表述间跳来跳去不落地 | 选定一个目标表述，解完再回；不并行试用多种 |
| 用"换工具"逃避真正的困难 | 如果所有表述下都难，困难不在表述而在问题本身 |
