# 摄动渐进求解

## 描述

复杂非线性问题无法一步解决时，先解简化版的标称问题，将真实偏差视为小扰动逐次修正，每次修正后评估近似边界是否被突破。

## 使用场景

### 何时使用

- 直接求解非线性问题失败或代价过高
- 需要将复杂问题拆解为"理想情况 + 修正项"
- 迭代优化中每次只做小幅调整
- 评估近似方案的适用范围（"这个线性化在什么条件下成立？"）
- 需要给出"由粗到精"的渐进改进路径

### 何时不使用

- 问题已经可以直接精确求解
- 不存在可解的简化版本（无法构造标称问题）
- 扰动不是"小"的——偏离量与标称量同数量级
- 精度要求不需要修正（简化版已足够）

## 输入输出

### Input

| 字段 | 类型 | 说明 |
|------|------|------|
| problem | object | 原始问题描述，包含方程/约束/目标 |
| complexity_factors | list | 导致问题复杂的因素清单（非线性、耦合、变系数、随机干扰等） |
| precision_requirement | number | 终端指标的可接受误差阈值 |

### Output

| 字段 | 类型 | 说明 |
|------|------|------|
| nominal_solution | object | 标称问题的解 |
| perturbation_corrections | list | 各阶修正量 |
| validity_range | object | 近似成立的参数范围 |
| final_solution | object | 渐进解（标称 + 修正） |

## 指令

### 1. 构造标称问题

关闭原问题的所有复杂因素，得到一个可精确求解的简化版。简化手段：

| 原问题复杂因素 | 简化方式 |
|---------------|---------|
| 非线性方程 | 在工作点附近线性化 |
| 变系数 | 冻结系数为常值 |
| 耦合项 | 暂时解耦/忽略 |
| 随机干扰 | 取期望值（确定性化） |
| 时变边界条件 | 固定为标称值 |

验收：标称问题必须可解。若简化后仍不可解，继续放宽。

### 2. 建立一阶摄动方程

将真实解写为标称解 + 小偏差：`x(t) = x₀(t) + δx(t)`，代入原方程，展开到一阶，略去二阶及以上项，得到一阶摄动方程。注意：摄动方程通常是变系数线性方程，标称解作为时变参数进入系数矩阵。

### 3. 求解一阶修正

用标称问题中已算出的伴随函数/灵敏度，预估偏差对最终指标的影响。选择最易实现的方法：

- **伴随方程法**：逆向积分一次即可评估任意初始偏差的影响
- **直接积分法**：对每个偏差单独正向积分摄动方程
- **解析灵敏度公式**：偏导有闭式时直接代入

### 4. 判断近似边界

检查摄动假设是否成立：

- ‖δx(t)‖ / ‖x₀(t)‖ < ε（偏差量级远小于标称量级）
- 一阶修正的贡献 > 二阶项（一阶近似有效）
- 终端指标偏差在可接受范围内

若近似边界被突破：在突破点将当前实际值设为新标称基准，回到步骤 1；或保留标称，展开到二阶摄动。

### 5. 迭代至收敛

重复"标称 → 一阶修正 → 评估"循环，直到满足精度要求，输出渐进解。

### 操作边界

- **定性策略**：直接推理——"先忽略非线性项求解，再考虑偏差方向和量级"
- **需要具体修正量**：必须写代码做符号微分或数值摄动展开
- **禁止**：假装手算了一阶摄动修正量；给出精确修正值而无计算过程

## 失败策略

| 失败场景 | 处理方式 |
|---------|---------|
| 标称问题不可解 | 继续放宽简化条件直到可解；若仍不可解，考虑切换数学表述（衔接 `transforming-representations-for-equivalence`） |
| 摄动假设不成立（偏差与标称同量级） | 在突破点重设标称基准重新迭代；若多次重设仍不收敛，放弃摄动法改用数值方法 |
| 临界情况（特征根实部为零） | 一阶近似失效，改用非线性分析方法（中心流形、分岔分析） |
| 一次展开到高阶导致计算爆炸 | 严格限制在一阶；不够时再谨慎做二阶；禁止跳到更高阶 |

## 示例

### 示例 1：弹道摄动修正

**问题**：导弹在标准大气下算出理想弹道，需评估风偏和推力偏差的影响。

1. 构造标称问题：无风、标准大气、额定推力下的弹道 → 精确解出 x₀(t)
2. 建立摄动方程：δx 的变系数线性方程，x₀(t) 进入系数矩阵
3. 伴随方程法逆向积分一次，得到任意初始偏差对落点偏差的影响
4. 检查 ‖δx‖/‖x₀‖ < 5%，成立 → 输出修正后落点

### 示例 2：非线性电路工作点分析

**问题**：含非线性元件的电路，需求工作点附近的小信号响应。

1. 标称问题：将非线性元件在工作点处线性化，求直流工作点
2. 摄动方程：小信号等效电路（线性）
3. 直接求解小信号增益
4. 检查信号幅度是否在线性化范围内 → 若超出，重新设定工作点
