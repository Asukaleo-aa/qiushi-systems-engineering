# 应用系统思维 — 输出格式模板

## 系统层次判定

```
系统层次：[物理系统 / 工程系统 / 产品系统 / 社会-技术系统]
母系统（如果有）：[这个系统服务于哪个更大的系统？]
层次选择依据：[为什么选这个层次？]
```

## 系统使命

```
系统使命：[一句话]
成功指标：[可量化标准列表]
关键约束：[不可违反的硬约束]
```

## 系统边界

```
系统内：[组分列表]
系统外（环境）：[环境因素列表]
交换接口：[输入列表] | [输出列表]
```

## 组分关联矩阵

```
组分A → [作用类型] → 组分B
组分B → [作用类型] → 组分C

关联矩阵：
        A    B    C    D
     A  -   +1    0   -1
     B  +1   -   +2    0
     C   0   -2    -   +1
     D  +1    0   -1    -
（+表示增强/依赖，-表示制约/冲突，0表示无直接关联，数值表示强度）
```

## 层次结构

```
哲学层（为什么）: [根本假设、指导原则]
理论层（是什么）: [核心原理、基本规律]
方法层（用什么）: [技术方法、分析工具]
执行层（怎么做）: [具体步骤、操作流程]
```

## 涌现特性

```
预期涌现特性：[列表]
潜在意外涌现：[列表]
触发条件：[对每个涌现特性的条件说明]
```

## 反馈结构

```
正反馈回路（放大）:
  A↑ → B↑ → A↑ （增强型）
负反馈回路（稳定）:
  C↑ → D↓ → C↓ （调节型）
关键延迟：[哪些反馈有显著滞后]
```
