# 原著依据

## 核心方法论来源

### 亚里士多德《形而上学》

第一原理（First Principles）概念的哲学源头：每个科学领域都存在一些最基本的、不可再推导的前提，一切推理都建立在这些前提之上。

### 笛卡尔《方法论》

系统怀疑的方法：将一切可怀疑的信念暂时搁置，直到找到不可怀疑的基点，再从基点出发重建知识体系。本 skill 的"剥离"步骤直接受此启发。

### 钱学森、宋健《工程控制论》第1章

> "每个系统的运动状态的特性与描述这个系统的微分方程的类型是有密切关系的。不但如此，微分方程的类型还确定了解决系统的工程问题的正确作法。"

钱学森不按"应用领域"（航空、化工、电气）分类控制系统，而是按描述系统的**微分方程基本类型**（线性/非线性、常系数/变系数、集中参数/分布参数）分类。方程类型是"第一性"——它决定了系统的全部行为特性和可用的分析方法。这种"从数学本质而非工程表象出发"的分类法，正是第一性原理思维在工程控制论中的集中体现。

## 与矛盾分析法的分工

`contradiction-analysis` 和本 skill 是 Layer 1 的双子星：

| | 第一性原理分析 | 矛盾分析法 |
|---|---|---|
| 追问 | "最基本的事实是什么？" | "核心的张力在哪里？" |
| 切入 | 向下挖到不可再分的根基 | 横向着眼于对立面的统一和斗争 |
| 产出 | 问题的基本元素和本质约束 | 主要矛盾及其主要方面 |
| 时序 | 先——先理解本质 | 后——在本质基础上找矛盾 |

典型流程：`analyzing-from-first-principles`（"这个系统本质上是什么"）→ `contradiction-analysis`（"它的主要矛盾是什么"）→ `investigating-before-deciding`（"去调查验证"）。
