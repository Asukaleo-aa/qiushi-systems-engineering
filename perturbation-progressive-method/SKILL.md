# 摄动渐进方法

> "实际弹道偏离标准弹道是很小的。因此，可以在标准弹道附近将运动方程作一次近似展开。"
> —— 钱学森、宋健《工程控制论》第13章

## 核心原则

**复杂非线性问题无法一步解决时：先解简化版的"标称问题"，将真实偏差视为"小扰动"逐次修正，每次修正后评估近似边界是否被突破。**

本方法源自航天工程中的弹道摄动理论：先计算理想弹道（无干扰、标准大气），再对实际偏差（风、密度变化、推力偏差）做一阶修正，在修正不"小"的地方重新设定标称基准。

> 详细原著依据见《工程控制论》第13.1—13.3节（摄动方程的建立）、第8章（轨道摄动法与极大值原理）。

## 不适用场景

以下情况**不需要**调用本 skill：
- 问题已经可以直接精确求解
- 不存在一个可解的简化版本（没有"标称"问题可构造）
- 扰动不是"小"的——偏离量与标称量同数量级
- 精度要求不需要修正（简化版已足够）

## 何时使用

- 直接求解非线性问题失败或代价过高
- 需要将复杂问题拆解为"理想情况 + 修正项"
- 迭代优化中每次只做小幅调整
- 评估近似方案的适用范围（"这个线性化在什么条件下成立？"）
- 需要给出"由粗到精"的渐进改进路径

## 方法流程

### 第一步：构造标称问题（第0阶近似）

将原问题的所有复杂因素关闭，得到一个**可精确求解**的简化版。

简化手段：
| 原问题复杂因素 | 简化方式 |
|---------------|---------|
| 非线性方程 | 在工作点附近线性化 |
| 变系数 | 冻结系数为常值 |
| 耦合项 | 暂时解耦/忽略 |
| 随机干扰 | 取期望值（确定性化） |
| 时变边界条件 | 固定为标称值 |

**验收标准**：标称问题必须可解。如果简化后仍不可解，继续放宽。

### 第二步：建立一阶摄动方程

将真实解写为标称解+小偏差：
```
x(t) = x₀(t) + δx(t)
u(t) = u₀(t) + δu(t)
```

代入原方程，展开到一阶，略去二阶及以上项，得到**一阶摄动方程**（线性微分/代数方程）。

**关键**：摄动方程通常是变系数线性方程——标称解 x₀(t) 作为"时变参数"进入系数矩阵。

### 第三步：求解一阶修正

用标称问题中已算出的伴随函数/灵敏度，预估偏差对最终指标的影响：

```
ΔJ ≈ ∫ ψ^T(t) · [摄动驱动项] dt
```

其中 ψ(t) 是伴随变量，由伴随方程 dψ/dt = -A^Tψ 从终端逆向积分得到。

**方法选项**（选最易实现的）：
- 伴随方程法：逆向积分一次即可评估任意初始偏差的影响
- 直接积分法：对每个偏差单独正向积分摄动方程
- 解析灵敏度公式：如偏导有闭式则直接代入

### 第四步：判断近似边界

检查摄动假设是否成立：
- ‖δx(t)‖ / ‖x₀(t)‖ < ε（偏差量级远小于标称量级）
- 一阶修正的贡献 > 二阶项（一阶近似有效）
- 终端指标偏差在可接受范围内

**如果近似边界被突破：**
- 在突破点将当前实际值设为**新标称基准**，回到第一步
- 或者：保留标称，展开到二阶摄动

### 第五步：迭代至收敛

```
标称 → 一阶修正 → 评估 → 满足精度？
                              ↓ 否 → 新标称 → 一阶修正 → ...
                              ↓ 是 → 输出渐进解
```

## 与其他skill的关系

**认知位置**：求解环。当精确求解不可行时启用。

**同环工具**：
- 标称问题的数学工具选择 → `multi-representation-equivalence-transform`
- 摄动多级修正需协调各阶改进 → `hierarchical-decomposition-coordination`
- 如摄动修正可形式化为优化 → `constrained-optimization`

**回连上游**：
- 摄动渐进本身是一种自适应策略 → `adaptive-robust-strategy`（提供自适应框架的理论基础）
- 摄动近似结果的验证遵循 `practice-cognition` 的实践检验原则
- 当多级修正遇到矛盾时回到 `contradiction-analysis` 确认主要矛盾

## ⚠️ 操作边界：何时做符号展开 vs 何时定性说明

| 问题类型 | 操作方式 |
|---------|---------|
| **定性策略** | 直接推理。"先忽略非线性项求解，再考虑它们造成的偏差方向和量级" |
| **需要具体修正量** | 必须写代码做符号微分或数值摄动展开（sympy / 有限差分） |

**绝不能做的事：** 假装手算了一阶摄动修正量；给出精确修正值而无计算过程。

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 标称问题选了不可解的 | 先简化到可解，再逐步加回复杂因素 |
| 不检查摄动假设是否成立就输出 | 必须在"偏差/标称"比值上设定阈值 |
| 一次展开到很高阶 | 先做一阶，不够再做二阶。高阶摄动极难 |
| 忽略了"临界情况"（特征根实部为零） | 此时一阶近似失效，需要非线性分析 |
